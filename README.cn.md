## :lemon: 用法

我建议全局安装该工具，这样无论你在做什么项目都可以使用它。但是如果你想在特定项目中使用它，你可以去掉 -g 标志
```sh
npm i -g css-learner
```

默认语言版本由node环境决定。终端输入css：
```sh
css
```
使用中文版本的CSS终端工具
```sh
csscn
```
使用英文版本的CSS终端工具
```sh
cssen
```
